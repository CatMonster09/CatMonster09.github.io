    • Introduction

    My Passion Project is about coding. Coding is a language that both humans and computers can understand. You have to follow syntax rules and put together the right symbols in the correct order to tell the computer what you want it to do. Coding is used to make software, apps, and websites... like this one!

    Programming Languages
    There are many programming languages in the world. Scratch is a programming language that helps kids understand and visualize how to code. It uses blocks instead of syntax. HTML (Hypertext Markup Language) is a programming language that is used for making documents made up of a bunch of tags. CSS (Cascading Style Sheets) is used for styling your webpage and making it look unique and original. JS (Java Script) makes a webpage interactive for the user. Including games and animations. These are just some of the many programming languages that you can use to code.

    Why Coding is Important

    If you know how to code it can help with creativity. It strengthens the ability to focus on small detail and adds to your problem solving skills. You can develop websites, video games, software, or anything that your imagination can take you to with coding. Many jobs require computer programming. Start learning now!
